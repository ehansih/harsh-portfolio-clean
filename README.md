# harsh-portfolio-clean

Personal portfolio site for **Harsh Vardhan Singh Chauhan** — Telecom Security · 5G · Product · AI.

Clean, minimal design inspired by modern product landing pages. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no build step.

## Stack

- HTML5 + CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (IntersectionObserver for scroll reveal)
- Google Fonts: Inter + IBM Plex Mono
- Formspree for contact form

## Design

- Near-black background (`#0c0c0c`)
- Lime accent (`#d4ff6e`)
- Full-width centered hero
- Stats strip
- 4-col pillars grid
- Vertical timeline
- 3-col project grid
- Contact form + channels

## Sections

- Hero — headline, tags, CTAs
- Stats — years, programs, countries, companies
- About — 4 capability pillars
- Experience — 16-year timeline (Ericsson → Jio → Nokia → Mavenir → Nokia)
- Projects — 6 open source repositories
- Principle — blockquote
- Contact — Formspree form + direct channels

## Deploy

Hosted on Vercel. No build step required — static HTML.

```bash
open index.html
```
